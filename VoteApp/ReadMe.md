
<h1 align="center"> Real-Time-Votes-backend </h1>



## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd dir
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```\
